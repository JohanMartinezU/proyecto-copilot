# Blog Técnico: Arquitectura de Software

**Autor:** Johan Ricardo Martinez Pedroza

## Descripción

Blog técnico estático implementado con HTML5, CSS3 y JavaScript vanilla. Contiene tres publicaciones sobre Arquitectura de Software, siguiendo estándares de desarrollo web moderno y mejores prácticas.

## Contenido

### Publicaciones

1. **¿Qué es la Arquitectura de Software?** (`posts/post1.html`)
   - Definición y conceptos fundamentales
   - Objetivos y componentes básicos
   - Importancia en desarrollo
   - Roles del Arquitecto de Software
   - Ejemplos en proyectos reales

2. **Estilos Arquitectónicos Más Utilizados** (`posts/post2.html`)
   - Estilo Monolítico
   - Arquitectura en Capas
   - Arquitectura Cliente-Servidor
   - Arquitectura MVC
   - Microservicios
   - Comparativa y recomendaciones

3. **Patrones Arquitectónicos Modernos** (`posts/post3.html`)
   - Clean Architecture
   - Arquitectura Hexagonal
   - CQRS (Command Query Responsibility Segregation)
   - Ventajas, desventajas y aplicaciones prácticas

## Características Técnicas

### HTML5 Semántico
- Etiquetas semánticas: `<header>`, `<main>`, `<article>`, `<section>`, `<footer>`
- Atributos ARIA para accesibilidad
- Meta tags de SEO
- Skip link para navegación por teclado

### CSS3 Moderno
- Sistema de variables CSS (custom properties)
- Flexbox y CSS Grid
- Mobile-first responsive design
- Media queries (768px y 480px)
- Transiciones suaves y hover effects

### JavaScript Vanilla
- Detección de página activa
- Scroll suave
- Sin dependencias externas

## Cómo Usar Localmente

```bash
# Opción 1: Servidor local con Python
cd proyecto-copilot
python -m http.server 8000
# Abre http://localhost:8000
```

## Despliegue en GitHub Pages

1. Sube el repositorio a GitHub
2. Ve a Settings → Pages
3. Selecciona `main / (root)` como fuente
4. Accede a: `https://TU-USUARIO.github.io/proyecto-copilot/`
